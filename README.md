# Blog Draft Writer Agent

Turn a blog idea into a polished markdown draft, ready to review and publish. Hand the agent an idea — title, summary, and outline — and dial in tone, length, audience, and voice. The agent writes a structured draft with a title, excerpt, and clean markdown body, optionally weaving in your reference content and citing the sources you supply.

## Capabilities

- Write a complete blog draft from a title, summary, and outline
- Adapt the tone, length, audience, and voice to match the brief
- Weave reference content and brand voice context into the draft
- Cite supplied sources inline where they support the argument
- Produce reusable draft content the rest of the blog pipeline can pick up
